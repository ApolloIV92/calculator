Calculator Project

This is a calculator project developed for The Odin Project. This calculator also takes numpad input. The "delete" key deletes characters, and the "backspace" key does a full erase. For numbers larger than 9,999,999,999,999 (the max width of the display area) the calculator will return the solution in scientific notation, formatted down to a maximum of7 decimal places to account for screen size. Thank youso much for looking, and please let me know of any bugs!
